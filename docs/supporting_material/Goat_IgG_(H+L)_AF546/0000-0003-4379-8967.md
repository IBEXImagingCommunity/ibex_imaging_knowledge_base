# Additional Notes

<a name="notes"></a>
1. This antibody can be used to evaluate unconjugated goat primary antibodies in a wide range of tissues. Be careful when using unconjugated primary antibodies as IBEX protocol does not strip antibodies. Place antibodies in first cycle and use direct conjugates in later cycles. This conjugate does not dye inactivate within 15 minutes of 1 mg/ml LiBH4 treatment. Additional dye inactivation conditions need to be tested.
